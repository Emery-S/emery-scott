# Emery Scott Portfolio

## Deploy to Vercel (Free)

1. **Push to GitHub**
   - Create new repo at github.com/new
   - Upload this folder

2. **Deploy on Vercel**
   - Go to vercel.com, sign up with GitHub
   - Click "New Project" → Import your repo
   - Click Deploy
   - Done! Live at `your-project.vercel.app`

3. **Custom Domain** (optional)
   - Settings → Domains → Add your domain
